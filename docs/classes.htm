Classes and Functions

The classes and functions in the SCATMECH library can be divided into seven catagories:

  • Polarization
  • Optical Properties of Materials
  • Free-Space Scattering Models
  • Grating Models
  • Surface Scattering Models
  • Property Models
  • Utility

Polarization

The following classes and functions enable calculations of polarimetric quantities by providing some of the more common representations for polarization states, their transfer functions, and common elements.

  • class MuellerMatrix
  • class StokesVector
  • class JonesMatrix
  • class JonesVector
  • Common polarization elements and states
